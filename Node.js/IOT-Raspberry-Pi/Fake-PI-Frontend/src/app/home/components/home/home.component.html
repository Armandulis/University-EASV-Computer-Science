Overview

Backend Http -> save data to mongo DB and send new message to MQTT

Backend Socket -> Listen for when we need to receive new values from DB

Nest.js ( Backend ) - hosted on Heroku

Backend is responsible for saving new sensor values to MongoDb. Then it sends new Message to CloudMQTT. In the backend we also have set up socket.io, which will inform all clients about new sensor values.

Checkout backend
Backend Socket

In backend we listen for new messages from MQTT, once we receive a new message, we will emit/inform clients about new sensors data
